 初中英语智能教学助手

[![React](https://img.shields.io/badge/React-18-blue.svg)](https://reactjs.org/)
[![Vite](https://img.shields.io/badge/Vite-Project-646CFF.svg)](https://vitejs.dev/)

本项目是一款专为初中英语教师设计的轻量化智能教学辅助系统。不同于传统的标准化考试系统，本项目致力于适配**非标准化教学场景**，让 AI 技术真正下沉到日常作业批改、学生档案管理及个性化教学中。

 核心创新点
- **非标场景适配**：摆脱对特定答题卡和昂贵扫描仪的依赖，支持“随手拍”作业识别。
- **三段式重叠切割策略**：针对长图作业，通过算法自动切割与合成，提升识别准确率。
- **数据驱动闭环**：自动构建“批改-归档-再应用”链条，实现自动化的学生成长档案。
- **校验即教研**：将人工复核整合进错题整理过程，降低教师负担。

安全与隐私架构
项目针对第三方免费表单环境（Quickform）设计了一套“向下兼容”的高强度防御策略：
- **离线随机置换映射**：真实姓名与学生编号对应表仅保存在本地设备，云端数据完全脱敏。
- **非对称混淆处理**：数据在传输前进行混淆，物理隔离敏感信息。
- **多重身份复核**：引入密码 + 邀请码 + 设备绑定逻辑，确保数据访问安全。

技术栈
- **前端**：React 18 + Vite
- **AI 模型**：支持 Qwen3-VL 等主流大模型（通过 API 调用）
- **算法**：编辑距离算法（比对逻辑）、图像对比度增强与降噪算法
